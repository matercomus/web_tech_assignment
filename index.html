Report (Our experiences) testing HTML & CSS

Browsers: Google Chrome & FireFox

Our experiences working with Chrome was much better compared to FireFox, because it doesn't change certain things on the layout.

In FireFox the submit button changed to 'submit Query' and another difference was that in FireFox the list-style was absence from the unordered list.

Print style: was tested by removing all element (display: none), and we added incrementally the important parts of the website and check the layout (ctrl+p).

Screen style: every time we made a change to the style we checked if the correct/ desired changes was made in the browsers.

Handheld style: was tested by using the developer-tools in the browsers, to adjust/check the page on different screen size.
